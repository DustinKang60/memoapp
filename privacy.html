개인정보처리방침

작은앱공방(이하 "개발자")이 제공하는 메모장 앱(이하 "앱")의 개인정보처리방침입니다.

시행일: 2026년 4월 15일

1. 수집하는 개인정보

본 앱은 사용자의 개인정보를 수집하지 않습니다. 모든 메모 데이터는 사용자의 기기 내부에만 저장됩니다.

2. 개인정보의 이용 목적

본 앱은 서버로 데이터를 전송하지 않으며, 외부 서버에 사용자 정보를 저장하지 않습니다.

3. 위치 정보

일기 작성 시 날씨 정보를 불러오기 위해 사용자의 동의를 받아 위치 정보를 일시적으로 사용할 수 있습니다. 이 정보는 날씨 API 호출에만 사용되며 저장되지 않습니다.

4. 제3자 제공

본 앱은 사용자의 정보를 제3자에게 제공하지 않습니다.

5. 데이터 보관

모든 메모 데이터는 사용자의 기기 로컬 저장소(LocalStorage, IndexedDB)에만 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

6. 광고

본 앱은 광고를 포함하지 않습니다.

7. 어린이 개인정보 보호

본 앱은 만 13세 미만 어린이를 대상으로 하지 않으며, 어린이의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보처리방침 변경

개인정보처리방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 공지합니다.
